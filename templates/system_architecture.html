System Architecture Diagram

[Presentation Layer] -> [Application Layer] -> [Security Layer] -> [Data Layer]
                                  \-> [Blockchain Layer]

Presentation Layer: HTML/CSS/JS interfaces and interaction flow.

Application Layer: Flask routes for wallet and admin operations.

Security Layer: OTP auth, encryption, fraud checks, RBAC, lockout, logging.

Data Layer: SQLite tables for users, transactions, alerts, logs, and failed logins.

Blockchain Layer: Hash-linked blocks with chain verification.

Data Flow Diagram

User Action -> Flask Route -> Validation -> DB Update -> Blockchain Append -> Audit Log -> UI Response
